# AutoSync Brand Assets

## Colors
- Primary Blue: #0066ff
- Dark Background: #0f1729

## Logo Files
Place the following logo files here:
- `logo-icon.png` — Blue "A" icon only (square)
- `logo-horizontal.png` — Full "AUTOSYNC" horizontal logo with icon
- `logo-icon.svg` — SVG version of icon

## SVG Logo (inline)
The AutoSync logo consists of two overlapping triangles forming an "A" shape
with a curved gap between them, filled solid in #0066ff.

## Usage
- App header: icon only (24px)
- Vehicle spec page footer: icon + "Powered by AutoSync" text
- Shopify App Store listing: full horizontal logo
